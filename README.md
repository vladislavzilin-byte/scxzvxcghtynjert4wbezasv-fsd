
# IZ Booking v5 (Vite + React)

Функции:
- Личный кабинет клиента: профиль (Имя, Instagram, Телефон), список записей, счётчик активных, фильтр (Все/Активные/Отменённые), отмена записи.
- Бронирование до +24 месяцев вперёд.
- Модальные окна подтверждения/отмены.
- Админ может отменять записи; статус виден клиенту.
- Рабочие часы по умолчанию: 04:00–20:00.
- Логотип в шапке и футере. Тёмная тема. 

Старт:
```bash
npm i
npm run dev
```
