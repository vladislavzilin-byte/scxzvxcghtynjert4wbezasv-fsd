
# IZ Booking v7 (Vite + React)

Новые функции:
- Все записи создаются в статусе **pending** и требуют **подтверждения администратора** (даже если это сегодня).
- В разделе **Мои записи** добавлена кнопка **Обновить** (со спиннером), чтобы перезагружать список без обновления страницы.
- Переключатель языка (LT / RU / EN) с простым i18n, язык сохраняется в localStorage.

Остальное:
- Админка: поиск, фильтры, сортировка, экспорт CSV, спиннер при обновлении, подтверждение/отмена заявок.
- Календарь: бронирование на +24 месяца, модальные окна.
- Тёмная тема IZ HAIR TREND, логотип в шапке и футере.
- Рабочие часы по умолчанию: 04:00–20:00.

Запуск:
```bash
npm i
npm run dev
```
